Desktop Speech To Text

Pure Whisper. Now with translate and rewrite modes.

NanoWhisper is a minimal desktop speech-to-text app for macOS, Windows, and Linux. Press a shortcut, speak, and paste text back into the app you were already using.

What's New
Translate Mode
Default shortcut: Ctrl+T. Record once and paste both the transcript and the translation with automatic line breaks.
Modify Mode
Default shortcut: Ctrl+O. Select text, speak your instruction, and replace the original selection in place.
Focus Restore
Recording now tries to preserve the original typing target so paste returns to the previous app and caret position more reliably.

Core Features

One Shortcut

Start and stop recording with a global hotkey. No extra navigation, no large UI.

Auto-Paste

Transcribed or rewritten text goes straight back to the active application.

History

All results are stored locally with audio files, so you can retry later with a different model.
